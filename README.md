## Requirements

- An Arch Linux-based distribution (Arch Linux, EndeavourOS, CachyOS, Manjaro, etc.)
- Hyprland
- Git

## Installation

Clone the repository:

```bash
git clone https://github.com/zheny-creator/Hyprland-rice.git
cd Hyprland-rice
```

Make the installation script executable:

```bash
chmod +x install.sh
```

Run the installer:

```bash
./install.sh
```
